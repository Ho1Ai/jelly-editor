# JELLY Text Editor

Jelly text editor is a small text editor for config management written in C++.  
Mostly the Jelly is [in](https://github.com/Ho1Ai/in) clone, rewritten in C++. Also it adds raw mode

## How to use

Base commands:  
- out - output everything  
- ins - insert string  
- afl - add free line  
- rml - remove line  
- cfn - change file name  
- rma - remove text (area)  
- rmf - remove text (from)
